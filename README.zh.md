# 子弹笔记助手 Obsidian 插件

[中文](README.zh.md) | English

一个用于可视化子弹笔记项目任务的 Obsidian 辅助插件，支持日历和甘特图视图。

## 功能特性

### 项目管理
- **项目列表**：显示所有项目及其基本信息
- **项目详情**：查看详细的项目信息和相关材料
- **项目筛选**：按名称、状态等筛选项目

### 任务管理
- **任务列表**：显示项目下的所有任务
- **任务详情**：查看任务详情、链接和相关项
- **任务层级**：可视化任务层级关系（@L1、@L2、@L3）

### 事项管理
- **事项列表**：显示所有事项及其关联的任务和项目
- **事项详情**：查看事项描述和日期
- **事项筛选**：按日期、项目、任务等筛选事项

### 日历视图
- **日/周/月视图**：以日历格式显示每日工作事项
- **事项标记**：在日历上标记工作事项
- **交互功能**：点击日期查看详细事项

### 甘特图视图
- **项目甘特图**：显示项目时间线和任务进度
- **任务依赖关系**：可视化任务依赖关系
- **时间范围选择**：支持查看不同时间范围的甘特图
- **进度追踪**：标记任务完成状态和进度

## 安装方法

1. **手动安装**：
   - 从 GitHub 仓库下载最新版本
   - 将文件解压到 Obsidian 插件目录
   - 重新加载 Obsidian
   - 在 Obsidian 设置中启用插件

2. **从 Obsidian 插件市场安装**：
   - 打开 Obsidian 设置
   - 导航到"社区插件"
   - 搜索 "子弹笔记"
   - 点击"安装"然后"启用"

## 配置说明

1. **数据目录**：设置子弹笔记数据目录的路径
2. **默认年份**：设置要显示的默认年份

## 使用指南

### 打开视图
- **命令面板**：使用 Obsidian 的命令面板打开不同视图
- **侧边栏图标**：点击侧边栏的日历图标，在菜单中选择日历视图、甘特图视图或项目视图
- **快捷键**：为快速打开视图分配快捷键

### 视图导航
- **项目视图**：点击项目查看其任务和详情
- **日历视图**：使用日历控件在不同日期之间导航
- **甘特视图**：使用时间线控件在不同时间范围之间导航

### 事项交互
- **任务事项**：点击任务事项查看其详情和相关项
- **日历事项**：点击日历事项查看其详情
- **甘特图事项**：点击甘特图事项查看其详情

## 数据格式

插件期望的子弹笔记数据格式如下：

```markdown
## [项目名称]
> [项目描述]
> [材料名称]: [链接]

### 工作任务
[任务名称] #任务
[任务链接](https://example.com)
[事项 1] @[日期]
[事项 2] @[日期]

[任务名称 2] #任务
[任务链接](https://example.com)
[事项 1] @[日期]

[任务名称 3] #任务 @L1
[事项 1] @[日期]
[事项链接](https://example.com)
```

### 任务层级标记
- `@L1` - 父任务（第 1 级）
- `@L2` - 子任务（第 2 级）
- `@L3` - 孙任务（第 3 级）

## 开发指南

### 环境要求
- Node.js
- npm
- Obsidian

### 设置步骤
1. 克隆仓库
2. 运行 `npm install` 安装依赖
3. 运行 `npm run dev` 启动开发服务器
4. 创建符号链接到 Obsidian 插件目录

### 构建
运行 `npm run build` 构建生产版本插件
